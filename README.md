# Deep-Reinforcement-Learning-Trading-Agent-for-European-Equity-Markets
Training an AI Trader using PPO to Optimize Risk-Adjusted Returns

## Reinforcement Learning Approach

An RL agent learns trading behavior through interaction.

* The system contains:

1: Agent

2: The AI trader

3: Environment

4: Financial market

5: Reward

6: Trading profit adjusted for risk


### This project implements a reinforcement learning based trading system for European equity markets.

    The agent learns optimal trading policies using Proximal Policy Optimization (PPO).
